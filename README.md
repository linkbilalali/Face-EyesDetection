Face and Eye Detection System
This project is a computer vision application that detects faces and eyes in real-time. It uses Python and the OpenCV library with Haar Cascade Classifiers.

Features
Real-time detection via webcam or video files.

Simultaneous face and eye tracking.

Visual bounding boxes for detected features.

Optimized for stable performance.

Technologies
Python 3

OpenCV

NumPy

How to Setup
Install OpenCV and NumPy.

Ensure the Haar Cascade XML files are in the project folder.

Run the main python script to start detection.
